# AURORA — Panel
# Мощный инструмент управления прокси на базе Xray-core

## Быстрый старт

```bash
git clone https://github.com/aurora/panel.git
cd panel
cp .env.example .env
# Заполнить .env (пароли, секреты)
docker compose up -d
```

## Требования

- Docker Engine 24+ & Docker Compose v2
- Сервер: 2 CPU, 4 GB RAM минимум
